\doxysubsection{Detailed Description}
A graph class. In the context of this program this is going to represent the STCP network.

Due to the design choices made throughout the project, this class is a mix of the theoretical classes\textquotesingle{} \char`\"{}abstract\char`\"{} graph and a context-\/aware graph tailored to the STCP network.
